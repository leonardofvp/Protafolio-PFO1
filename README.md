📌 Portafolio - Práctica Formativa Obligatoria 1

📖 Descripción del Proyecto

Este proyecto consiste en la creación de un sitio web personal tipo portafolio, desarrollado con HTML y CSS. El objetivo es presentar información personal, habilidades técnicas, películas favoritas y un formulario de contacto. Además, se aplican buenas prácticas de maquetado, diseño responsivo y uso de herramientas como Google Fonts y GitHub Pages para su publicación.
